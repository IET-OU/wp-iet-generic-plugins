# IET-OU / wp-iet-generic-plugins

A collection of generic WordPress plugins, including:

* ` [tagcloud] ` shortcode
* ` [wp_query] ` shortcode
* Simple Embed
* Simple Menu
* IET attribution
* IET custom style
* _... And others!_

## Install .. test

```sh
composer install && composer npm-install
composer test
```

Work-in-progress:

```sh
composer cs && composer semi
composer fix
```
